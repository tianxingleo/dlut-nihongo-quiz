# AI 功能使用指南

## 功能概述

本应用集成了 AI 辅助功能，可以帮助你：

1. **AI 详细解析** - 获取题目的详细解析，AI 会分析题目考查的知识点、逐项分析选项、指出原解析可能的错误
2. **AI 问答** - 自由提问，AI 会结合学科特点回答你的问题

## 配置步骤

### 1. 获取 API Key

支持以下 AI 服务提供商：

| 提供商 | API 地址 | 模型名称 |
|--------|----------|----------|
| DeepSeek | https://api.deepseek.com | deepseek-chat |
| OpenAI | https://api.openai.com | gpt-4o-mini |
| 其他兼容服务 | 自定义 | 自定义 |

### 2. 在应用中配置

1. 打开应用，进入 **设置** 页面
2. 找到 **AI 助手** 区域
3. 启用 AI 功能
4. 填入你的 API Key
5. 确认 API 地址和模型名称（通常使用默认值即可）
6. 点击 **测试连接** 验证配置是否正确
7. 点击 **保存配置**

### 3. 使用 AI 功能

#### 在做题时使用

1. 进入做题界面
2. 答题后，点击 **AI 解析** 按钮获取详细解析
3. 或点击 **AI 问答** 按钮进行自由提问

#### 功能特点

- **流式响应** - AI 回答会实时显示，无需等待完整响应
- **上下文感知** - AI 会根据题目所属学科（日语/历史/党史/军事理论）提供针对性解析
- **错误纠正** - 如果原题库解析有误，AI 会指出并给出正确解释
- **扩展知识** - AI 会提供相关知识点扩展和记忆技巧

## 安全说明

- **API Key 安全** - 你的 API Key 存储在浏览器本地的 IndexedDB 中，不会上传到任何服务器
- **本地调用** - 所有 AI 请求直接从你的浏览器发送到 AI 服务提供商，不经过中间服务器
- **不提交代码** - API Key 不会被提交到代码仓库，`.env` 文件已被 `.gitignore` 排除

## 常见问题

### Q: 为什么 AI 解析没有出现？

A: 请检查：
1. 是否已在设置中启用 AI 功能
2. API Key 是否有效
3. 网络连接是否正常

### Q: AI 解析和原解析哪个更准确？

A: AI 解析仅供参考。建议结合原解析和 AI 解析综合判断，以教材和老师讲解为准。

### Q: 支持哪些 AI 服务？

A: 支持所有 OpenAI 兼容格式的 API，包括 DeepSeek、OpenAI、Claude API（通过代理）等。

### Q: 如何更换 AI 服务？

A: 在设置中修改 API 地址和模型名称即可，无需重新启用 AI 功能。

## 技术细节

- 使用 OpenAI 兼容的 Chat Completions API
- 支持流式响应（SSE）
- 最大 Token 数默认 2048，可在设置中调整
- 温度参数默认 0.7，影响回答的创造性（0=确定性，2=创造性）
